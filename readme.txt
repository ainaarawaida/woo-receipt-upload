== Installation ==

Make sure that you already have WooCommerce plugin installed and activated.

**Step 1:**

- Login to your *WordPress Dashboard*
- Go to **Plugins > Add New**
- Search **receipt upload for WooCommerce**

**Step 2:**

- **Activate** the plugin through the 'Plugins' screen in WordPress.

**Step 3:**

- Go to **WooCommerce > Settings > Payments**
- Click **Manage**

**Step 4:**

- Fill in your Account No Account Bank Name and your name for offlinevpayment
- Make sure the 'Enable this payment gateway' is ticked.
- Click **Save** to save changes.

**Step 5:**

- Go to your woocommerce > order > action > receipt.
- You can check any order with receipt upload by showing action icon(clicking Option tob bar menu )
